# web

This folder is designated for web application code.

Live on https://promosencegrupp9-4bbcc.web.app/

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run dev
```

or

```
npm run start
```

### Build and deploy to Firebase

```
npm run build
```

then

```
firebase deploy
```
